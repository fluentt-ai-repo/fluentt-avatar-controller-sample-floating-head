# v0.2.1 - Talking Animation / Energy Matching 제거 (Idle 전용 경량화)

**릴리스 날짜**: 2026-02-26

## 주요 변경사항

### Talking Animation 시스템 제거

FloatingHead 패키지를 idle 전용으로 경량화했습니다.
전체 기능(Talking + Energy Matching)은 AvatarControllerSample v0.2.0으로 이전되었습니다.

### 제거된 기능

#### 1. Talking Animation
- `FluentTAvatarControllerFloatingHead.TalkingAnimation.cs` 삭제
- `talkingAnimations` 필드 제거
- `SetTalking()` 호출 제거 (OnSentenceStarted, OnSentenceEnded)
- `talking_override_0.anim`, `talking_override_1.anim` 삭제

#### 2. Energy Matching
- `FluentTAvatarControllerFloatingHead.EnergyMatching.cs` 삭제
- `enableEnergyMatching`, `energyBlendRatio` 필드 제거
- `InitializeEnergyMatching()`, `UpdateAudioRMSForEnergyMatching()` 호출 제거

#### 3. Editor UI 축소
- Talking Animations 섹션 제거 (HelpBox + PropertyField)
- Energy Matching 섹션 제거 (토글 + 슬라이더 + HelpBox)
- SwapBuffer 자동 설정에서 Talking 0/1 상태 제거

### 수정된 동작

- `OnSwapSlotReady()`: Idle 그룹만 처리 (Talking 분기 제거)
- `OnSentenceStarted()`: Talking 전환 없이 콜백만 유지
- `OnSentenceEnded()`: Idle 복귀 없이 emotion 리셋만 유지
- `Start()`: `InitializeTalkingAnimations()`, `InitializeEnergyMatching()` 호출 제거
