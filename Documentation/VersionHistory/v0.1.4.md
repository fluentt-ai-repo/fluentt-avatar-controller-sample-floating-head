# v0.1.4 - Eye Blink 레이어 변경

**릴리스 날짜**: 2026-01-30

## 주요 변경사항

### Eye Blink 레이어 인덱스 변경
- `BLINK_LAYER_INDEX`를 2에서 0으로 변경
- Talkmotion SDK v0.6.0의 내부/외부 레이어 분리에 대응
  - 내부 레이어 (음수): FluentTAvatar/Timeline 시스템 전용
  - 외부 레이어 (0 이상): 사용자/컨트롤러용

## 변경 이유

Talkmotion SDK v0.6.0에서 Face/Head 애니메이션이 분리되면서:
- Face 애니메이션: 레이어 -2 (LAYER_FACE_ANIMATION)
- Head 애니메이션: 레이어 -1 (LAYER_HEAD_ANIMATION)

기존 Eye blink가 레이어 2를 사용하여 Head 애니메이션과 충돌하던 문제가 있었습니다.
새로운 레이어 시스템에서는 외부 코드(컨트롤러)가 0 이상의 레이어만 사용하도록 규칙이 정해졌습니다.

## API 변경

```csharp
// Before
private const int BLINK_LAYER_INDEX = 2;

// After
// Internal layers (negative): -3=Base face, -2=Face animation, -1=Head animation
// External layers (0+): Available for user/controller use
private const int BLINK_LAYER_INDEX = 0;
```

## 의존성

- Talkmotion SDK v0.6.0 이상 필요
