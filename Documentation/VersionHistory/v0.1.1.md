# v0.1.1 (2024-12-10)

> 초기 릴리스. Floating head 아바타용 최적화된 바디 애니메이션 제어 컴포넌트를 제공합니다.

## 주요 기능

- **Eye Blink Animation**: 랜덤 간격으로 눈 깜빡임 애니메이션
- **Look Target Control**: 가상 타겟과 가중치 블렌딩을 사용한 동적 시선 제어
- **Emotion Tagging**: 텍스트 기반 감정 감지 및 표정 제어
- **Server Motion Tagging**: 서버 기반 모션 태그 응답
- **Body Animation**: 상태 머신 기반 바디 애니메이션 시스템

## 설계 목적

Talkmotion SDK는 **립싱크와 얼굴 애니메이션에만 집중**합니다. AvatarControllerSampleFloatingHead는 이와 분리하여 **Floating head 아바타의 바디 애니메이션 제어**를 담당합니다.

### 아키텍처

```
┌─────────────────────────────────────────┐
│      TalkMotion SDK                     │
│  ✅ 립싱크 (얼굴 blendshape)              │
│  ✅ Animator Parameter 전달              │
│     - IsTalking, OnTalkingStart, ...   │
└─────────────────────────────────────────┘
              ↓ Parameters
┌─────────────────────────────────────────┐
│ AvatarControllerSampleFloatingHead      │
│          (본 패키지)                     │
│  ✅ Eye blink (눈 깜빡임)                │
│  ✅ Look target (시선 제어)               │
│  ✅ Body animation (최소한의 바디 제어)    │
│  ✅ Emotion control (감정 표현)          │
└─────────────────────────────────────────┘
```

## AvatarControllerSample vs FloatingHead

| 특징 | AvatarControllerSample | FloatingHead |
|------|------------------------|--------------|
| 대상 | 전신 아바타 | Floating head 아바타 |
| 바디 애니메이션 | 풍부한 바디 모션 | 최소한의 바디 제어 |
| 최적화 | 일반 | Head-only 최적화 |
| 사용 시나리오 | 게임, VR 경험 | UI, 챗봇, 키오스크 |

## 사용 방법

1. `FluentTAvatarControllerFloatingHead` 컴포넌트를 아바타 GameObject에 추가
2. Inspector에서 아바타 참조 및 설정 구성
3. 컨트롤러가 FluentTAvatar 컴포넌트와 자동 통합

## 호환성

- Unity 2022.3 이상
- Talkmotion SDK (`com.fluentt.talkmotion`) 필수

## 의존성

- Talkmotion SDK (`com.fluentt.talkmotion`) v0.1.0 이상

## 초기 릴리스 기능

- Eye blink animation system with random intervals
- Look target control with virtual targets and weighted blending
- Emotion tagging system for text-based facial expressions
- Server motion tagging integration
- Body animation state machine system
- Editor tools for avatar configuration
