# v0.2.4 - One-Shot Motion 기능 추가 및 Editor 성능 최적화

**릴리스 날짜**: 2026-03-31

## 주요 변경사항

### 1. One-Shot Motion API

외부 시스템(Flutter 브릿지 등)에서 특정 모션을 즉시 1회 재생할 수 있는 API를 추가했습니다.

#### 개별 모션
- `PlayOneShotMotion(motionId)` — 등록된 모션 1회 재생 후 자동 Idle 복귀
- `StopOneShotMotion()` — 즉시 중단 및 Idle 복귀
- `IsOneShotMotionPlaying()` — 재생 상태 확인
- `GetOneShotMotionIds()` — 등록된 motionId 목록 조회

#### 그룹 모션 (랜덤 루프)
- `PlayOneShotMotionGroup(groupId)` — 그룹 내 weight 기반 랜덤 재생, 중지될 때까지 반복
- `IsOneShotGroupLooping()` — 그룹 루프 상태 확인
- 연속 중복 재생 방지 (2개 이상 entry 시 직전 clip 제외)

#### TalkMotion 우선
- TalkMotion 재생 중 one-shot 요청 거부
- TalkMotion 시작 시 one-shot/그룹 자동 중단

#### 콜백
- `onOneShotMotionStarted(motionId)` / `onOneShotMotionEnded(motionId)` UnityEvent 제공

### 2. Inspector — One-Shot Motion 섹션

- 개별 모션 등록 리스트 (motionId, clip, blendWeight)
- 그룹 등록 리스트 (groupId, entries with weight/blendWeight)
- 검증: 중복 ID, 빈 ID, null clip 경고
- Play 모드 디버그: 드롭다운 + Play/Loop/Stop 버튼

### 3. Editor 성능 최적화

- `serializedObject.FindProperty()` 49개를 `OnEnable` 캐싱으로 전환 (매 프레임 → 1회)
- `new GUIContent()` ~25개를 `static readonly` 캐싱
- Validation용 `Dictionary`/`List` 인스턴스 재사용 (`.Clear()`)
- `AnimatorController.layers` 접근 캐싱 (배열 복사 방지)

### 4. Inspector UI — 아코디언 스타일 전환

수평 Toolbar 탭을 lilToon/Shuriken 스타일 아코디언(접이식 헤더) UI로 변경하여 FluentTAvatarEditor와 동일한 UX 제공.
