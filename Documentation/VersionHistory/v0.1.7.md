# v0.1.7 - 에디터 UI 리네이밍 및 감정 리셋 토글

**릴리스 날짜**: 2026-02-24

## 주요 변경사항

### 1. 에디터 탭 리네이밍

Inspector 탭 이름을 기능의 실제 역할에 맞게 변경했습니다.

| 기존 | 변경 후 |
|---|---|
| Emotion Tagging | Text Emotion Detection |
| Emotion Motion Mapping | Gesture Animation |

### 2. C# 필드명 리네이밍

에디터 UI와 일관되도록 필드명도 변경했습니다. `FormerlySerializedAs`로 기존 시리얼라이즈 데이터 호환성을 유지합니다.

| 기존 필드명 | 새 필드명 |
|---|---|
| `enableClientEmotionTagging` | `enableTextEmotionDetection` |
| `emotionMotionMappings` | `gestureMappings` |
| `EmotionMotionMapping` (클래스) | `GestureMapping` (클래스) |

### 3. 감정 리셋 토글 추가 (`enableAutoEmotionReset`)

마지막 문장 종료 시 제스처 애니메이션을 Idle로 자동 복귀하는 기능을 토글로 제어할 수 있습니다.

- **기본값**: `true` (기존 동작 유지)
- **Off 시**: 마지막 문장이 끝나도 현재 제스처 애니메이션이 계속 재생됨
- Animator Controller에 `emotionReset` 트리거 및 0.3초 블렌드 전환 추가

### 4. 데드 코드 제거

- `ProcessServerMotionTagging()` — `OnServerMotionTag` 콜백으로 대체된 레거시 메서드
- `GetLayerIndex()` — `LayerWeightController`로 대체되어 미사용
- `UpdateTimelines()` / `LateUpdateTimelines()` — 빈 메서드 및 `Update()` 호출부
