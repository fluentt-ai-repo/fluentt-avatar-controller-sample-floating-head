# v0.1.3 - Head Control 비활성화 시 Eye Control 동작 안함 버그 수정

**릴리스 날짜**: 2026-01-15

## 주요 변경사항

### 버그 수정

#### Enable Head Control 비활성화 시 BlendWeightFluentt Eye Control 동작 안함 수정

- **문제**: `enableHeadControl`이 `false`일 때 `eyeControlStrategy`가 `BlendWeightFluentt`인 경우 눈 추적이 전혀 동작하지 않음
- **원인**: `InitializeLookTarget()`에서 `headAimConstraint == null` 체크가 무조건 수행되어, Head Control이 꺼져있을 때 (headAimConstraint가 null로 설정됨) early return되어 `lookTargetController`가 초기화되지 않음
- **해결**: `headAimConstraint` null 체크를 `enableHeadControl`이 `true`일 때만 수행하도록 조건 수정

### 수정 전
```csharp
if (headAimConstraint == null)
```

### 수정 후
```csharp
if (enableHeadControl && headAimConstraint == null)
```
