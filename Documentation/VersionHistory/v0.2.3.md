# v0.2.3 - 런타임 Rig 생성/삭제 기능 및 Inspector UI 추가

**릴리스 날짜**: 2026-03-03

## 주요 변경사항

### 1. 런타임 Rig Setup/Destroy API

Play 모드에서 Animation Rigging 계층을 동적으로 생성하고 삭제할 수 있는 3개의 public 메서드를 추가했습니다:

#### `SetupLookTargetRigAtRuntime()`
- RigBuilder, Rig, MultiAimConstraint, VirtualTarget을 런타임에 생성
- Animator bone 자동 탐색 (`FindLookTargetTransforms()`)
- Head tracking + Eye tracking (Transform / TransformCorrected / BlendWeightFluentt 전략별 분기)
- `RigBuilder.Build()` 호출로 PlayableGraph 활성화
- `enableLookTarget = true` 설정 및 `InitializeLookTarget()` 호출

#### `DestroyLookTargetRigAtRuntime()`
- LookTargetController 비활성화 및 참조 해제
- `RigBuilder.Clear()` + layers 제거
- TargetTracking의 Constraint/Rig 컴포넌트만 제거하고 GameObject는 비활성 상태로 유지
  - Animator 내부 PropertyStreamHandle 바인딩이 남아있어 GameObject를 삭제하면 "Could not resolve" 경고가 무한 발생하는 문제 방지
- VirtualTargets 정리 및 직렬화 필드 참조 초기화

#### `RebuildRig()`
- `RigBuilder.Build()` 수동 호출 유틸리티

### 2. Inspector UI — Runtime Rig Control

Play 모드에서만 표시되는 Inspector 버튼 3개 추가:
- **Setup Rig** — `SetupLookTargetRigAtRuntime()` 호출
- **Destroy Rig** — `DestroyLookTargetRigAtRuntime()` 호출
- **Rebuild** — `RebuildRig()` 호출

### 3. Animation Rigging 미설치 시 Stub 구현

`#else` 블록에 3개 메서드의 stub 구현 추가 (경고 로그 출력).
