# v0.1.2 (2026-01-08)

> Editor 코드 구조 개선 및 partial 클래스 분리

## 변경사항

### Editor 코드 리팩토링

- **Partial 클래스 분리**: `FluentTAvatarControllerFloatingHeadEditor`를 탭별 partial 클래스로 분리
  - `FluentTAvatarControllerFloatingHeadEditor.cs` - 메인 (OnInspectorGUI, Helper Methods)
  - `FluentTAvatarControllerFloatingHeadEditor.DefaultAnimation.cs` - Default Animation 탭
  - `FluentTAvatarControllerFloatingHeadEditor.LookTarget.cs` - Look Target 탭 + Rig Setup
  - `FluentTAvatarControllerFloatingHeadEditor.ServerMotionTagging.cs` - Server Motion Tagging 탭
  - `FluentTAvatarControllerFloatingHeadEditor.EyeBlink.cs` - Eye Blink 탭

### 코드 품질 개선

- **Helper 메서드 추가**: Reflection 패턴 단순화
  - `GetFieldValue<T>()` - private 필드 값 읽기
  - `SetFieldValue()` - private 필드 값 설정
  - `FindOrCreateVirtualTargetsContainer()` - VirtualTargets 컨테이너 관리
  - `FindOrCreateAvatarVirtualTargetGroup()` - 아바타별 가상 타겟 그룹 관리
  - `FindAvatarVirtualTargetGroup()` - 가상 타겟 그룹 검색

- **상수화**: 반복 사용되는 값들을 상수로 추출
  - `PrivateInstance` - BindingFlags 상수
  - `LogPrefix` - 로그 메시지 접두사

- **Dictionary 패턴 적용**: `AutoFindEyeBlendShapes`의 if-else 체인을 Dictionary 매핑으로 개선

## 개선 효과

- 탭별 파일 분리로 코드 가독성 향상
- 특정 탭 수정 시 해당 파일만 읽으면 되어 토큰 사용량 절약
- Helper 메서드로 코드 중복 제거
- 유지보수성 향상

## 호환성

- Unity 2022.3 이상
- Talkmotion SDK v0.1.0 이상
- 기존 API 호환 유지 (breaking changes 없음)
