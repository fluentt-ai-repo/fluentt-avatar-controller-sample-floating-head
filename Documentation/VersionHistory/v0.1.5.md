# v0.1.5 - 컴포넌트 자동 설정 개선

**릴리스 날짜**: 2026-02-05

## 주요 변경사항

### 1. Animator Controller 자동 할당

컴포넌트가 처음 추가될 때 패키지 내 `FluentTAvatarFloatingHeadController.controller`가 자동으로 할당됩니다.

**동작**:
- `Reset()` 메서드에서 `animatorController`가 null일 때 자동 로드
- 에디터 전용 기능 (`#if UNITY_EDITOR`)

### 2. 기본 Idle 애니메이션 클립 자동 할당

컴포넌트 추가 시 `defaultIdleAnimationClip`에 샘플 애니메이션이 자동 할당됩니다.

**추가되는 클립**:
- `idle_sample.anim` → `defaultIdleAnimationClip`

### 3. 기본 Blink Clip 자동 생성

컴포넌트 추가 시 `blinkClip`이 자동으로 생성됩니다.

**동작**:
- ARKit eyeBlinkLeft/Right 블렌드쉐이프 기반
- 기본 타이밍: 0s(0) → 0.06s(100) → 0.08s(100) → 0.18s(0)

### 4. enableLookTarget 기본값 변경

`enableLookTarget` 기본값을 `true`에서 `false`로 변경했습니다.

**변경 이유**:
- Look Target 기능은 Animation Rigging 설정이 필요함
- 초기 설정 없이 활성화 시 오류 발생 가능
- 명시적으로 설정 후 활성화하도록 유도
