# Changelog

All notable changes to this package will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-12-10

### Added
- Initial release of FluentT Avatar Controller Sample
- Eye blink animation system with random intervals
- Look target control with virtual targets and weighted blending
- Emotion tagging system for text-based facial expressions
- Server motion tagging integration
- Body animation state machine system
- Editor tools for avatar configuration
